条件式 - コンソール画面で確認

See the Pen 条件分岐 by ryuji takimoto (@ryuzie) on CodePen.

変数ageに20と19を比べて条件を満たしているので、コンソール画面に「成人しています。」と表示されます。

条件には、下記の比較演算子を使って条件を作成します。
A < B　AはBより小さい時
A > B　AはBより大きい時
A <= B　AはB以上の時
A >= B　AはB以下の時
A == B　AとBが同じ時
A != B　AとBが同じではない時
A === B　AとBが完全に同じ時 ※文字同士など、型まで同じ時

条件文を書いてみましょう

定数 itemAを作成し、200を代入する
定数 itemBを作成し、200を代入する
itemAとitemBを比べて、同じ場合という条件文を書き
同じ条件の場合に、「同じ数値です」とコンソール画面に表示させること。
