01で記載された条件は、19より大きい数値の場合でした。
数値が条件に合わない場合はコンソール画面に表示されないため、if文の内容はスキップされたことが分かります。
次に、条件に合わなかった場合には、ことなった内容を表示させる為にif文に、条件に合わなかった場合のelseを追加しましょう。

書き方
if(条件){
条件に合う場合
}
else{
条件に合わなかった場合
}

条件式 - コンソール画面で確認

See the Pen 条件分岐02 by ryuji takimoto (@ryuzie) on CodePen.

定数ageに10が入っているため、条件に合わずelse内にある「未成年です。」と表示されます。

条件文を書いてみましょう

定数 walletを作成し、2000を代入する
定数 itemAを作成し、2980を代入する
walletとitemAを比較し、
購入可能（walletの数値以下）であれば「購入することができます」とコンソール画面に表示。
購入不可であれば「購入することができません」とコンソール画面に表示。
walletは2000なので、「購入することができません」と表示されたら、定数walletの値を変更して
「購入することができます」も確認しておきましょう。

Point

wallet（財布）とitemA（商品）を比べて購入出来るかを比較するので
財布より商品の値段が小さい場合と考えること。
※財布の中身がピッタリある時は購入可能なので条件の記号に気をつけましょう。
