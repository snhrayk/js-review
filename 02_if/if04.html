価格帯を決める際、例えば100〜500の間の様な時や、
マイナスの年齢や120歳以上の時など、条件を省きたい場合
複合した条件が必要です。
100以上「かつ」500以下
0より下「または」120以上

Aの条件 かつ Bの条件 の様に２つの条件どちらの条件も満たす場合は、&& で区切ります
Aの条件 または Bの条件 の様に２つの条件どちらか一方の条件が合う場合は、|| で区切ります

if文をネスト（親子関係:A条件の中にB条件）にして、記述した場合、2つの条件を満たす条件になります 


        if(age >= 0){
          //0以上の場合
          if(age <= 6){
            //6以下の場合　つまり、0以上6以下の条件に満たされる数値の時だけ実行される
          }
        }

        if(age >= 0 && age <= 6){
          //上のネストと同じ条件です。0以上6以下の条件に満たされる数値の時だけ実行される
        }
