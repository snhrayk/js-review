for文を使って、2の段の掛け算をコンソールで表示してみましょう。

for文を使って、2の段の掛け算を出力します。
表示内容は、
「2」「4」「6」...「20」となるようにコンソール画面に表示させましょう。

Point

変数iは繰り返しで使う変数のため、変数に値を代入してはいけません。


        NG）実行される度に変数iに3が代入され、繰り返しが終わらなくなります
          for(let i=0; i<3; i++){
            i = 3;
          }

        OK)変数iに1を足したものをnoに代入するため、変数iの値は変わらない
          for(let i=0; i<3; i++){
            const no = i + 1;
          }
