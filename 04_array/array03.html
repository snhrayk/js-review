配列その3 - 配列の末尾に追加 push

配列に後からデータを追加で入れたい場合、
配列名.push(追加したい内容)
で、配列の末尾にデータを後から追加することが可能です。

配列の作成

配列 web を作成し、
WD1A / WD2A を、各要素に入れて、コンソール画面に表示すること。
その後、WD3Aを末尾に追加し、改めてコンソール画面にWD3Aが追加されたかを表示すること。

Point

要素が2つある配列に、配列名[2] = "aaa" と書いても末尾にデータが入ります
pushの場合は「最後」なので、要素の数に関係なく末尾を指定することが可能です。
