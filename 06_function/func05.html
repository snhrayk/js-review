関数 その4 - 無名関数

関数作成時には、関数名を記入し作成を行いましたが、
function (){ }の様に関数名がないものを、「無名関数」といいます。
関数名がありませんのであとで呼び出す事ができない変わりに、その場限りのインスタントな関数です。
clickイベント等の時に記載している
() => { } これも無名関数の1つで、アロー関数と言います。
今後、モダンなJavaScript開発をする上で、よく使われるものになりますので、合わせて覚えておくと良いです。

Point

通常の関数（無名関数）とアロー関数（es6から追加）は、書き方が短くなったものではなく、細かな振る舞いが異なります。
JavaScriptは歴史の古い言語のため、古くからある言語機能とes6から追加されたものなど混同します。
今後JavaScriptのフレームワークなど、モダンな開発へとさらに学習を進みたい場合、新しく追加された機能や各種違いなどを
掘り下げて理解をすすめましょう。
